# 🔐 MegaSelf Authentication Server

Сервер аутентификации для MegaSelfCodeLoader с управлением пользователями и подписками.

## 🚀 Деплой на Render

### Автоматический деплой:
1. Этот репозиторий уже настроен для автоматического деплоя на Render
2. При каждом push в main ветку сервер автоматически обновляется

### Настройка переменных окружения в Render:
1. Зайди в свой проект на Render
2. Перейди в Environment
3. Добавь переменную:
   - `ADMIN_KEY` = `твой_секретный_ключ_2025`

### URL сервера:
После деплоя твой сервер будет доступен по адресу:
`https://loader-auth-server-xxxx.onrender.com`

## 🎯 Тестовый пользователь

Автоматически создается тестовый пользователь:
- **Username:** `testuser`
- **Password:** `testpass`
- **Подписка:** 1 год

## 📊 Endpoints

- **Главная:** `/`
- **Админ-панель:** `/admin`
- **Health check:** `/api/health`
- **Регистрация:** `POST /api/auth/register`
- **Вход:** `POST /api/auth/login`

## 🔧 Использование в лоадере

В коде лоадера замени URL на свой:
```cpp
const std::wstring AUTH_SERVER_URL = L"https://твой-сервер.onrender.com";
```

## 🛠️ Локальная разработка

```bash
npm install
npm start
```

Сервер запустится на http://localhost:3000

## 📝 Логи

Все действия логируются:
- Регистрации пользователей

- Административные действия

- Хеширование паролей с bcrypt
- Rate limiting для защиты от брутфорса
- HWID привязка (один аккаунт = один компьютер)
- Валидация всех входящих данных
